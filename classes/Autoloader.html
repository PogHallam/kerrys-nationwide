Autoloader.php : 22

Auto-load classes for the plugin.

USAGE:

  • Drop this class and its containing directory into the plugin root directory.

  • Call this class from your plugin like so:

       require_once PLUGIN_PATH . 'lib/Autoloader.php';
   Autoloader::_instance(*BASE NAMESPACE*); *

  • Note the BASE NAMESPACE argument is the top-level namespace of your plugin. It is removed before converting namespace to directory structure. e.g. "Vendor\MyPlugin" or just "MyPlugin"
